About Don

Don Odibat is the lead engineer and founder of Don System Holdings. With a focus on Zero-Knowledge Architecture, Don specializes in building high-utility web tools that function without server-side data persistence.

His work on TellMyLocation.com challenges the industry standard of aggressive data harvesting, proving that essential utilities like GPS and Weather can be delivered securely using client-side technologies (HTML5 Geolocation API and PWA standards).
