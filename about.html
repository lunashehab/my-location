About TellMyLocation

TellMyLocation.com is a technical utility designed for Privacy-First Positioning. In an ecosystem where many applications persistently track movement history, we provide an alternative: a stateless, client-side tool that utilizes standard hardware sensors to provide high-accuracy data without server-side storage.

The Mission

Founded by Don Odibat under Don System Holdings, our mission is to democratize access to high-precision location data without the privacy trade-off. We believe that knowing "Where am I?" is a fundamental right, and you shouldn't have to surrender your personal data to a tech giant just to answer that question.

The Architecture of Privacy

We utilize a Client-Side Architecture to ensure data minimization. This is a deliberate engineering choice that separates us from 99% of location apps:

  • Local Execution: The JavaScript code that calculates your position runs entirely inside your browser's memory (RAM).
  • Direct Hardware Access: We interface directly with the navigator.geolocation API standard, allowing your browser to communicate with your device's GPS chip.
  • Ephemeral Sessions: There is no database. There are no user accounts. When you close the tab, the data instance is cleared instantly.

The Technology Stack

Leaflet.js

An open-source JavaScript library for mobile-friendly interactive maps. It allows us to render lightweight map tiles without the tracking bloat of heavier map engines.

OpenStreetMap (OSM)

Our map data is sourced from OSM, a community-driven project. This ensures that our maps are free from corporate bias and constantly updated by local contributors.

HTML5 Geolocation API

The W3C standard for location access. It provides the "handshake" between our website and your device's GNSS chip, ensuring secure and permitted access only.

Progressive Web App (PWA)

Our site creates a "Service Worker" cache. This means after your first visit, the core code is stored on your device, allowing the tool to load even if you lose cell service.
