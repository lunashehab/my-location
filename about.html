About TellMyLocation

Transparency in Geolocation

TellMyLocation.com is a technical utility designed for Privacy-First Positioning. In an ecosystem where many applications persistently track movement history, we provide an alternative: a stateless, client-side tool that utilizes standard hardware sensors to provide high-accuracy data without server-side storage.

The Architecture of Privacy

We utilize a Client-Side Architecture to ensure data minimization:

  • Local Execution: The JavaScript code that calculates your position runs entirely inside your browser's memory (RAM).
  • Direct Hardware Access: We interface directly with the navigator.geolocation API standard, allowing your browser to communicate with your device's GPS chip.
  • Ephemeral Sessions: There is no database. There are no user accounts. When you close the tab, the data instance is cleared.

Technical Specification: W3C Geolocation Standard

We adhere to W3C High-Accuracy protocols. When permitted, we request a PositionOptions object with enableHighAccuracy: true. This instructs the device to prioritize satellite trilateration over generic Wi-Fi triangulation, aiming for the highest accuracy available to the device hardware (typically 5-20 meters).

How It Works: Multi-Source Positioning

To provide a fast and reliable fix, modern devices utilize a hybrid positioning system. Our tool displays the aggregate data from:

  1. GNSS (Global Navigation Satellite Systems): Direct signals from GPS, GLONASS, Galileo, or BeiDou satellites.
  2. Wi-Fi Positioning System (WPS): In urban environments where satellite line-of-sight is blocked, devices may triangulate position relative to known Wi-Fi access points.
  3. Cellular Trilateration: Uses signal timing from nearby cell towers to approximate location when other methods fail.
